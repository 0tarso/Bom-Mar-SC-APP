## Sobre

- Este projeto foi desenvolvido para facilitar o acesso à informação sobre a qualidade da água das praias do litoral catarinense.  
- Atualmente essas informações são disponibilizadas por órgãos públicos, porém com menor usabilidade.  
- O app consome diferentes APIs para cálculo de rota, clima e marés, utilizando um backend em Python.
