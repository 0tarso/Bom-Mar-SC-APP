## Sobre
- Este projeto foi desenvolvido para facilitar o acesso à informação sobre a qualidade da água das praias do litoral catarinense. 
- Atualmente essas informações são disponibilizadas por orgãos públicos. Informações essas que são utilizadas no app mas trazendo uma melhor usabilidade para o usuário.
- O app consome diferentes API's para calculo de rota, clima e marés disponibilizadas em um backend em python.
